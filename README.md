## 📌 About the Project

This is a **frontend clone of Zerodha’s landing page**, built using **HTML & CSS**.
The goal of this project is to practice **UI design, layout structuring, and styling** by recreating a real-world website.

---

## ✨ Features

* 🎯 Clean and minimal UI
* 📌 Fixed navigation bar
* 📊 Hero section with CTA
* 💼 Trust & features section
* 💰 Pricing section
* 📚 Varsity (education) section
* 📄 Detailed footer
